# 🌍 International Tip Calculator

## 📌 Overview
This project is a web-based **International Tip Calculator** built using **HTML, CSS, and JavaScript**.  
It allows users to calculate tip amounts, total bill, tax, and currency conversion dynamically.

---

## 🚀 Features

- ✅ Enter bill total and calculate tip instantly
- ✅ Adjustable tip percentage using slider (0–100%)
- ✅ Automatic tax calculation (11%)
- ✅ Tax Exempt option (for ITMD 541 requirement)
- ✅ Currency conversion:
  - USD ($)
  - Euro (€)
  - Indian Rupee (₹)
- ✅ Real-time updates (no refresh needed)
- ✅ Input validation for incorrect values
- ✅ Disabled output fields (auto-calculated)
- ✅ Responsive design (works on mobile + desktop)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
